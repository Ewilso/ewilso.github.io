Interactive Demonstration

How to use:
Start by thinking of a 4-digit number, consisting only of the numbers 1 to 6 (0 is excluded). The Algorithm will take a guess at what the number is and it is your job to respond. This takes the format of the mastermind peg system. For each digit that is correct and in the right position, type "B", and for each digit that is correct but in the wrong position, type "W". Hit return when you have formulated this and the Algorithm will calculate it's next guess. It should take no more than 5 guesses to work out your number.


When Finished:
When the Algorithm has finished, either type REL into the input box or reload the page, either of these will start the Algorithm again.
